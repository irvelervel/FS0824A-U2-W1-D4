CSS TRANSITIONS

La proprietà "transition" in CSS applica una transizione tra due valori di una proprietà, oppure addirittura di più proprietà insieme.

È sostanzialmente possibile "rallentare" l'applicazione di un nuovo valore per una proprietà, lasciando al browser il compito di interpolare tutti i valori intermedi.

La proprietà "transition" solitamente si utilizza per l'effetto di hover
